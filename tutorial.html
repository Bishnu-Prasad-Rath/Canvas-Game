Canvas Drawing Basics

HTML5 Canvas provides a powerful way to draw graphics using JavaScript. You can draw shapes, lines, curves, text, and images.

// Basic setup
var canvas = document.getElementById('tutorialCanvas');
var ctx = canvas.getContext('2d');

// Drawing a rectangle
ctx.fillStyle = 'blue';
ctx.fillRect(50, 50, 100, 75);

Understanding Radians

In canvas, angles are measured in radians, not degrees. The relationship is:

0° = 0 rad
90° = π/2 rad
180° = π rad
270° = 3π/2 rad
360° = 2π rad

Remember: 0 radians starts at the 3 o'clock position!

Text on Canvas

You can render text on canvas with various fonts, sizes, and styles.

// Drawing text
ctx.font = '30px Arial';
ctx.fillStyle = 'purple';
ctx.fillText('Hello Canvas!', 50, 50);

// Outlined text
ctx.font = 'bold 40px Georgia';
ctx.strokeStyle = 'darkblue';
ctx.strokeText('Outlined Text', 50, 100);

Bezier Curves

Bezier curves are defined by context points and control points that shape the curve.

// Quadratic Bezier curve
ctx.beginPath();
ctx.moveTo(50, 200);
ctx.quadraticCurveTo(150, 50, 250, 200);
ctx.stroke();

// Cubic Bezier curve
ctx.beginPath();
ctx.moveTo(50, 200);
ctx.bezierCurveTo(100, 50, 200, 50, 250, 200);
ctx.stroke();

Images on Canvas

You can draw images onto canvas, either from the page or loaded dynamically.

// Drawing an image
var img = new Image();
img.onload = function() {
  ctx.drawImage(img, 50, 50, 200, 200);
};
img.src = 'image.jpg';
